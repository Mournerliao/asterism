## What to build

在列表之外提供一个**可切换的第二视图**：把本人向量确定性降维到 2D 的语义星图。同批 star 每次打开位置一致（地图肌肉记忆）；分层渲染（远景看密度 / 区域轮廓，拉近显个体节点，永不同渲数千点）；星是克制的实体节点，仅选中 / 邻近 / 命中时点电光蓝；检索 = 点亮一条路径（命中升起、其余沉底）而非过滤。列表视图是它的 a11y 等价路径。为下一票预留「区域层」。

**prototype 先行**：降维算法（UMAP / PCA / t-SNE）的可复现性与 Canvas / WebGL 分层阈值先用 `/prototype` 收口，结论带回本票。

## Acceptance criteria

- [ ] 星图是与列表**并列、可切换**的第二视图，**不是**浮在内容下的背景层。
- [ ] 确定性降维：同一批向量每次投影到稳定的 2D 坐标（可复现）。
- [ ] 分层渲染 + 视口裁剪 / 聚合，大库**不同时渲染数千点**；远景密度、近景个体。
- [ ] 视觉严守 Graphite Glass：默认克制近素描，仅选中 / 邻近 / 命中点电光蓝 `#2563EB` 与玻璃材质，**无 glow、无噪点**。
- [ ] 检索在星图上表现为「点亮路径」，是与隐形混合搜索**同一套排序**的空间化表达。
- [ ] 列表视图作为键盘 / 读屏等价路径始终可用。
- [ ] en / zh-CN、reduced-motion、无 WebGL 优雅降级；`pnpm lint / typecheck / test / build` 全绿；更新 `knowledge/state/*` 与 `logs/`。
