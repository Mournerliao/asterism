# Empty state action hierarchy

Date: 2026-07-15

## Context

Collections 与 Tags 在数据为空时，同时于页头和空状态主体显示相同的 primary action，形成两个视觉权重相同、目的完全重复的入口，削弱了首次创建路径的层级。

## Changes

- 数据为空时隐藏页头创建按钮，仅保留空状态主体中的首次创建主操作。
- 已有集合或标签后恢复页头创建按钮，保持日常追加操作的可达性。
- 标签仅因搜索无结果而进入局部空态时，继续保留页头创建按钮，因为底层数据并非空集合。
- 在 UI/UX 契约中新增空状态操作模式，约束同一 primary action 不得在页头与空状态重复出现。

## Verification

- Collections / Tags 空数据状态均只有一个主操作。
- 非空状态仍可从页头创建集合或标签。
- 标签搜索无结果时页头创建入口仍可用。
